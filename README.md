# Data Structures in C++

## Project Description
Implementation of classic data structures in C++ with tests covering edge cases.

### Implemented Data Structures
- **Singly Linked List**  
- **Doubly Linked List**  
- **Binary Search Tree (BST)**  

---

## Features & Test Coverage

### Singly Linked List
- Add elements to the front and back (`push_front`, `push_back`)  
- Remove elements (`pop_front`, `pop_back`)  
- Clear the list (`clear_list`)  
- Edge case tests: operations on an empty list  

### Doubly Linked List
- Add elements to the front, back, and middle (`pushFront`, `pushBack`, `insertAt`)  
- Remove elements (`popFront`, `popBack`, `removeAt`)  
- Clear the list (`clear`)  
- Edge case tests: removing from a non-existent index  

### Binary Search Tree
- Insert elements (`insert`)  
- Search elements (`search`)  
- Remove elements (`remove`)  
- Find minimum and maximum (`minimum`, `maximum`)  
- Traversal: DFS and BFS  
- Edge case tests: removing non-existent elements, empty tree operations  


## How to run:
1. Clone the repository:<br>
  `git clone <repo_url>`
2. Navigate to the project folder:<br>
  `cd Data_Structures`
3. Compile the program:<br>
  `g++ *.cpp -o program.exe `
4. Run the executable:<br>
  `./program.exe `
